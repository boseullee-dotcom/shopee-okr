🔥 Firebase 연결이 필요합니다

이 대시보드는 팀이 함께 실시간 편집할 수 있도록 Firebase Realtime Database를 사용합니다.
리더가 1회 설정하면 모든 팀원은 별도 작업 없이 바로 사용할 수 있습니다.

방법 1 (권장): 리더가 firebase-config.js 파일에 Firebase 설정을 입력하고 GitHub repo에 업로드 → 모든 팀원에게 자동 적용됨.
→ README_사용법.md PART 1 참고
방법 2 (임시): 아래 ⚙️ 설정 버튼을 눌러 본인 브라우저에만 임시 적용 (다른 PC에서는 다시 입력 필요).

🔥 Firebase 설정

1. Firebase 콘솔에서 프로젝트 생성
2. 좌측 메뉴 Build → Realtime Database → Create Database (Singapore 권장, Start in test mode)
3. 좌측 톱니바퀴 Project settings → 하단 Your apps → Web (</> 아이콘) → 앱 등록 → 표시된 firebaseConfig 객체를 복사
4. 아래에 붙여넣고 저장
